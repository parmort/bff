# BFF

A terminal-based curses filebrowser. WIP.

## Building

Ensure `curses` is installed, along with `cmake` and `make`. Run `make` to
compile. `make clean` cleans build files.

## Mappings

| Key | Description |
|-----|-------------|
| `:` | Enter command mode |

## Commands

| CMD | Impl | Description |
|-----|------|-------------|
| `:q` | yes | Exit the program. |
